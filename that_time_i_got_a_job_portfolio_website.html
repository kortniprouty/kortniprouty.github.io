Project: That Time I Got a Job   •   Tools: RPG Maker MZ   •   Year: 2024

This small RPG explores work and humor in short-form gameplay. The screenshots below show UI, exploration, dialogue, and in-game scenes. Designed with a black and crimson brand in mind.

Design Intent

The game focuses on clear visual storytelling with readable UI, distinct character portraits, and a consistent color palette. This portfolio page highlights UI decisions, level layout, and art direction.

Role & Responsibilities

  • Level/event design using RPG Maker MZ
  • UI layout and menu design
  • Sprite placement and scene composition
  • Playtesting and iteration

Tools

RPG Maker MZ — tilemaps, event scripting, character sprites, and menu customization.
